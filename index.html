AuthPass

Free and Open Source password manager for Android, iOS and MacOS. Compatible with KeePass (kdbx 3 right now).


Source code hosted on github, together with releases for MacOS.

Download on GitHub

Secure your Passwords

  • Load/Decrypt kdbx 3.x using password
  • Search through stored credentials
  • Desktop (Mac): Keyboard shortcuts for Search, Copy, Navigation, etc.
  • Desktop & Mobile: Copy & paste support
